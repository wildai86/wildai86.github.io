核心优势：支持微信、银行卡等国内便捷支付方式，可订阅 ChatGPT Plus、Claude Pro 等海外AI服务；老用户可保留原有账号及余额，用于抵扣订阅费用；新用户无需实名制，降低隐私顾虑，操作门槛低。

## 三、订阅流程指南（分新用户、老用户）

订阅流程整体简洁，分为「新用户注册订阅」和「老用户余额兑换/直接订阅」两种场景，全程按页面提示操作即可，无需特殊网络配置，具体步骤如下：

### 3.1 新用户注册及订阅流程（无历史 WildCard 账号）

1. 访问核心官方直达链接 <https://bewild.ai?code=MASK>，点击页面「注册」按钮，选择邮箱注册（无需实名制），输入邮箱地址并设置登录密码，完成账号注册。
  
2. 注册成功后，登录账号，进入订阅页面，选择需要订阅的AI服务（目前主推 ChatGPT Plus，后续将逐步增加 Claude Pro、Cursor 等服务）。
  
3. 上传对应AI服务的 Access Token（以 ChatGPT 为例，需提前获取自身 ChatGPT 账号的 Access Token，用于平台完成代订阅操作），上传完成后，平台将生成 Stripe 支付链接。
  
4. 点击支付链接，选择支付方式（微信或银行卡），微信支付需输入姓名后扫码完成支付，银行卡支付按页面提示操作即可；支付完成后，平台将自动完成订阅绑定，无需用户手动操作后续步骤。
  
5. 订阅成功后，可在账号后台查看订阅期限、服务状态，后续如需续费，重复步骤 2-4 即可。
  

### 3.2 老用户订阅流程（有历史 WildCard 账号及余额）

老用户可直接使用原有 WildCard 账号登录，利用历史余额兑换或抵扣订阅费用，具体步骤如下：

1. 访问核心官方直达链接<https://bewild.ai?code=MASK>点击「登录」，输入原有 WildCard 账号（邮箱/手机号）及密码，完成登录（账号及余额均已保留，无需重新注册）。
  
2. 登录后，页面将自动展示历史余额兑换页面，点击「兑换」按钮，选择需要订阅的套餐（支持1个月至1年不等的套餐，可根据自身余额及需求选择）。
  
3. 余额抵扣规则：若余额充足，可直接全额兑换；若余额不足，可通过微信/银行卡补足差额后完成兑换，解决余额闲置问题（每个账户仅可兑换1次，兑换通道将在45天后关闭，请及时操作）。
  
4. 兑换完成后，无需额外上传 Access Token（若系统提示需补充，按页面提示操作即可），平台将自动完成订阅绑定；若无需兑换，也可直接选择新订阅套餐，用余额抵扣部分费用后支付剩余金额。
